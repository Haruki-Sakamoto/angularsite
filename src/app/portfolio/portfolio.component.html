おみくじ

使用言語：HTML CSS JavaScript

基本的なJavascriptの構文であるMath.floor(Math.random)でランダムな数値を出し、それをもとに運勢の表示を変えるという簡易的なおみくじです。

また、配列の中の値の数を増やしたりif文を用いる事でおみくじの確率を変更する事も可能。

じゃんけん

使用言語：HTML CSS JavaScript

HTML内に関数をかき、ボタンがクリックされたときに処理が行われる。基本的なJavascriptの構文であるMath.floor(Math.random)でランダムな数値を出し、それをもとに勝敗をつける仕組みで動く簡易的なじゃんけんゲームになります。

一度仕組みを作ってしまえば、確実に勝てるじゃんけんゲームを作ることも可能です。
